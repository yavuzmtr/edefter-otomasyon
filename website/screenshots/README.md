# Ekran Görüntüleri Klasörü

Bu klasöre aşağıdaki ekran görüntülerini eklemeniz gerekmektedir:

## Gerekli Dosyalar

1. **dashboard.png** - Kontrol Paneli
   - İçerik: Ana dashboard, istatistikler, sistem durumu
   - Önerilen boyut: 1920x1200 px

2. **automation.png** - Otomasyon Merkezi  
   - İçerik: E-Defter Tam Otomasyon ekranı, ayarlar
   - Önerilen boyut: 1920x1200 px

3. **email.png** - E-posta Sistemi
   - İçerik: Şirket seçimi, email ayarları, gönderim logları
   - Önerilen boyut: 1920x1200 px

4. **backup.png** - Yedekleme Sistemi
   - İçerik: Yedekleme ayarları, aktivite raporları
   - Önerilen boyut: 1920x1200 px

5. **reports.png** - Raporlama Sistemi
   - İçerik: GIB raporları, filtreleme, Excel export
   - Önerilen boyut: 1920x1200 px

6. **monitoring.png** - GIB Klasör İzleme
   - İçerik: Klasör izleme, dosya kontrolleri, durum tablosu
   - Önerilen boyut: 1920x1200 px

7. **deadline-tracker.png** - E-Defter Berat Yükleme Takip
   - İçerik: Yaklaşan tarihler, gecikmiş işlemler
   - Önerilen boyut: 1920x1200 px

8. **company-management.png** - Şirket Yönetimi
   - İçerik: Şirket listesi, Excel şablonu kullanımı
   - Önerilen boyut: 1920x1200 px

## Ekran Görüntüsü Çekme İpuçları

1. **Temiz Veri Kullanın**: Test verileri profesyonel görünmeli
2. **Yüksek Çözünürlük**: En az 1920x1200 px boyutunda
3. **Format**: PNG formatı (kayıpsız sıkıştırma)
4. **Pencere Boyutu**: Tam ekran veya maksimize durumdayken
5. **Arayüz Temizliği**: Gereksiz popup veya bildirimler kapatılmalı

## Kullanım Alanları

Bu ekran görüntüleri şu alanlarda kullanılacak:
- ✅ Web sitesi (edefterotomasyon.com.tr)
- ✅ Ürün broşürleri
- ✅ Sunumlar ve demo materyalleri
- ✅ Sosyal medya paylaşımları
- ✅ E-posta kampanyaları
- ✅ Satış dokümanları

## Screenshot Araçları

**Önerilen Araçlar:**
- Windows'da: Snipping Tool / Snip & Sketch (Win + Shift + S)
- Gelişmiş: ShareX (ücretsiz, ekran kaydı da yapabilir)
- Profesyonel: Greenshot (ücretsiz, açıklama ekleme özelliği)
